## 개요

이 프로젝트는 HTML, CSS, 그리고 Vanilla JavaScript를 사용하여 만든 간단한 Todo List 웹 애플리케이션입니다. 사용자의 할 일 목록을 로컬 스토리지에 저장하여 브라우저를 닫았다 열어도 데이터가 유지됩니다.

## 주요 기능

- 새로운 할 일 추가
- 할 일 목록 표시
- 할 일 완료 표시
- 할 일 삭제
- 할 일 수정
- 할 일 카테고리 분류
- 로컬 스토리지를 이용한 데이터 저장

## 기술 스택

- HTML
- CSS
- Vanilla JavaScript
- Local Storage API
